# Movie Search Site
This is a simple movie search application where users can find information about
movies.

## Tools Used
- React
- Typescript
- RestfulAPIs

## Setup Instruction
- Cloning the Repo
```bash
git clone https://github.com/PetJs/trustcrow-frontend
```
- Installing the Dependencies
```bash
npm install
```
- Running it locally
```bash
npm run dev
```
